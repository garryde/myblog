意大利罗马学生健康卡申请(医保办理)指南

发表于 2019-09-17 更新于 2021-02-23 分类于 呆利那些事 阅读次数： Valine：
本文字数： 3k 阅读时长 ≈ 5 分钟

本文介绍关于注册意大利国家健康服务（SSN），办理健康卡(Tessera Sanitaria)，注册家庭医生等医疗服务的相关事宜

0. 要不要买医保？

大部分人购买医保是为了申请或续居留使用，若以此为主要目的，则不建议购买。目前罗马在办理居留时并不严格要求购买保险/医保，即使极个别情况被要求提供，也可以后补。

若以使用医疗资源为目的则强烈建议购买，即使使用私立医疗服务时也可享受优惠。例如心电图检测原价37欧元，使用家庭医生开具的单子只要12欧元。

任何情况下不建议购买九十多的那款商业保险，性价比极低。

1. 意大利国家健康服务（SSN）

在意大利的外国人只要有正式居留都有权使用意大利国家健康服务（SSN），注册后可享有与意大利公民同样的医疗服务。

意大利国家健康服务包括：

  1. 选择全科家庭医生，他会免费看诊，并向你建议你所需要的专科医生（牙医、整形外科医生等）
  2. 在公立或有授权的私立医院免费就诊
  3. 免费享有不间断值班医疗服务（guardia medica）
  4. 支付较少的检查费用（查血、放射检查等）

意大利罗马学生ISEE办理指南

发表于 2019-09-17 更新于 2019-09-24 分类于 呆利那些事 阅读次数： Valine：
本文字数： 1k 阅读时长 ≈ 2 分钟

1. ISEE 是什么

ISEE(经济状况等效指标)是一个家庭的济收入状况的评估证明，这份文件全意大利有效并 且是免费制作的。

2. ISEE 有什么用

可以用ISEE 申请经济帮助以及减免公共服务的费用（比如：生育津贴、助学金、经济帮助、学校食堂、 托儿所等…）。

3. ISEE 有效期

ISEE 的有效期是到次年的1月15日。

例如：你在2019年9月办理了ISEE ，有效期到2020年1月15日。

意大利罗马税号申请指南

发表于 2019-08-25 更新于 2021-03-16 分类于 呆利那些事 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 3 分钟

作为一个意大利留学生，税号涉及到生活的方方面面，例如：办理交通年卡，办理居留。本文仅适用于罗马的税号申请，其他城市亦可参考。

1. 税号是什么

Codice Fiscale（个人税号） 是一个16位的字母数字代码，由 Agenzia delle Entrate（国家税务局）免费发放以识别意大利人和外国居民。

意大利财政部在1974年发布一套算法，税号是根据姓名、性别、生日、出生地计算得出的，并非是申请之后才随机产生。

所谓的申请税号，即为到税务局注册该税号。所以大学的学生系统中有学生的税号，是根据个人资料计算得来，并不意味该税号已在税务局注册。
