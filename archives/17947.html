意大利罗马税号申请指南

发表于 2019-08-25 更新于 2021-03-16 分类于 呆利那些事 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 3 分钟

作为一个意大利留学生，税号涉及到生活的方方面面，例如：办理交通年卡，办理居留。本文仅适用于罗马的税号申请，其他城市亦可参考。

1. 税号是什么

Codice Fiscale（个人税号） 是一个16位的字母数字代码，由 Agenzia delle Entrate（国家税务局）免费发放以识别意大利人和外国居民。

意大利财政部在1974年发布一套算法，税号是根据姓名、性别、生日、出生地计算得出的，并非是申请之后才随机产生。

所谓的申请税号，即为到税务局注册该税号。所以大学的学生系统中有学生的税号，是根据个人资料计算得来，并不意味该税号已在税务局注册。

2. 办理地址

可以去罗马的任何一个Agenzia Delle Entrate办理，推荐市区交通相对较方便的一个，可以坐3号电车🚃前往。推荐早上去，否则极易面临长达数小时的排队等待。

Agenzia delle Entrate Direzione Provinciale I di Roma
Via Ippolito Nievo, 48, 00153 Roma RM, 意大利
+39 06 583191
https://goo.gl/maps/FpHzybriecR2

疫情须知！！

自2021年3月1日起，税务局仅接受官网预约办理业务，通常预约时间在两到三周后。

另外，刚来意大利税号可能尚未开通，开通税号需要预约，预约需要填写有效税号，无疑陷入死循环。可以选择让朋友帮忙预约，更建议进行在线办理。相关信息可点击：意大利罗马税号在线申请指南

3. 准备材料

3.1 证件/用品

  1. 护照原件
  2. 护照个人信息页复印件
  3. 签证页复印件
  4. 租房合同（可选）
  5. 带一只黑色笔用于填材料

有租房合同会进行税卡发放，没有则不会发放

3.2 申请表

该申请表现场会提供，可当场填写。亦可自己提前完成，加快办理速度。

（现场工作人员给完申请表后就会给第二个号，有可能申请表没有填完，但是已经叫号的情况，会比较仓促）

以下是意大利语版本的填写说明，英文版本的每个空的位置都是一样的。

注：没有意大利地址请按照上方填写，有意大利地址不填写图中QUADRO D，改为填写QUADRO C，内容为意大利地址。

4. 办理流程

全程会有两次叫号

第一次叫号

进门后右手边有一个取号机，若无工作人员，直接按第一个按钮取号即可，若有工作人员，告知 Codice Fiscale 即可，他会帮忙取号。第一次的号均在50、51窗口办理。

接待的工作人员会讲英文，告诉他是刚来罗马的新生，需要申请税号（Codice Fiscale），工作人员会给一张表格（若已经提前填好，把表格给工作人员看一下即可），一张新的号。外国人通常都给英文表格，如果你会意大利语亦可索取意大利语版本。

第二次叫号

第二次的办理窗口在里面工作区。看好对应的办理窗口，把护照，复印件全给工作人员即可。

办理完成后，工作人员会给一张税号单，至此，税号已经办理完成。以后有需要时，将工作人员返还的税号单复印上交即可。

5. 领取税卡

若提交住房合同并填写意大利地址，税卡(绿卡)将在两周后邮寄给你。税卡可在办理相关业务时出示，亦可在自动售卖机上购买香烟。
